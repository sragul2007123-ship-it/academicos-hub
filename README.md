# academicos — Starter Scaffold

This scaffold provides a minimal Next.js + TypeScript + Tailwind starter tailored for the academicos visual system.

Quick start

```bash
cd academicos-starter
npm install
npm run dev
```

What’s included

- Next.js 15 + TypeScript
- Tailwind CSS with Neo Aurora tokens
- Basic `HeroSphere` R3F placeholder
- `GlassCard` and `MagneticButton` components
- CSS design tokens in `src/styles/globals.css`

Notes

- The 3D hero is a minimal placeholder; replace with a polished React-Three-Fiber scene for the full effect.
- Use `three@0.152.2` for compat with `@react-three/drei` in this repo.
